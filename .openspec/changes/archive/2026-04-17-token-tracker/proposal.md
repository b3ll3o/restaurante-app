# Proposta: Token Tracker Plugin

## Problema
Não há visibilidade sobre o consumo de tokens durante o uso do OpenCode, dificultando o controle de custos e planejamento de uso.

## Solução Proposta
Instalar o plugin `opencode-token-tracker` para monitorar e exibir estatísticas de consumo de tokens.

## Impacto
- [ ] Breaking changes? Não
- [ ] Migração necessária? Não
- [ ] Novos dependencies? Sim (opencode-token-tracker)

## Alternativas Consideradas
- Manter controle manual de gastos - descartada por ser inviável
- Usar ferramenta externa de billing - descartada por não dar visibilidade em tempo real
